# Definition: sort arranges input lines alphabetically or numerically.
# Use it for: ordering data, removing duplicates, and comparing entries.

# Alphabetical sort
```
sort names.txt
```
# Reverse sort
```
sort -r names.txt
```
# Numeric sort
```
sort -n scores.txt
```
# Sort by 2nd column
```
sort -k2 file.txt
```
# Unique + sort
```
sort -u file.txt
```
